# i18n

> **Path:** `src/renderer/i18n/index.ts`, `src/renderer/i18n/locales/*.json`
> **Responsabilidade em uma frase:** Inicializar i18next com locales pt-BR e en.

## Responsabilidade
`main.tsx` importa `./i18n` antes do App, registrando recursos de traducao.

## API Publica
| Arquivo | Conteudo |
|---|---|
| `index.ts` | Configuracao i18next/react-i18next. |
| `locales/pt-BR.json` | Textos pt-BR. |
| `locales/en.json` | Textos en. |

## Fluxo Interno
Import side-effect inicializa i18n global para hooks/components que usem `react-i18next`.

## Erros e Edge Cases
- Muitas strings ainda estao hardcoded em componentes conforme codigo lido; i18n nao cobre toda UI.

## Side Effects
Inicializacao global i18next.

## Dependencias
`i18next`, `react-i18next`.

## Consumidores
Renderer.

## Testes
Sem teste direto.

## Observacoes / Dividas
Expandir cobertura de chaves se internacionalizacao virar requisito real.
